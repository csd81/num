Files:
  1. gaussian_elimination.m (Driver script)
     - Demonstrates all Gaussian elimination methods
     - Example systems with expected solutions
     - Comments on use cases for each method

  2. gaussian_elimination_no_pivoting.m (Function)
     - Basic Gaussian elimination without pivoting
     - Fixed zero-pivot check (before inner loop)
     - Use case: Well-conditioned systems
     - Returns: solution vector x

  3. gaussian_partial_pivoting.m (Function)
     - Gaussian elimination with row pivoting
     - Selects max |A(i,k)| at each step
     - Use case: General linear systems
     - Returns: solution vector x
     - Source: fixed from foelem/rgauss.m

  4. gaussian_complete_pivoting.m (Function)
     - Gaussian elimination with row and column pivoting
     - Most numerically stable method
     - Tracks variable permutations
     - Use case: Ill-conditioned systems
     - Returns: solution x, column permutation vector
     - Source: fixed from foelem/tgauss.m

  5. gauss_jordan_elimination.m (Function)
     - Gauss-Jordan with partial pivoting
     - Solution read directly from RREF
     - Use case: Single right-hand side, educational
     - Returns: solution vector x

  6. matrix_inverse.m (Function)
     - Matrix inversion via Gauss-Jordan
     - Augmented system [A | I] → [I | A^(-1)]
     - Includes verification A*A^(-1) = I
     - Condition number check (rcond)
     - Returns: inverse matrix
